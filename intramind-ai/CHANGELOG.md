# Changelog

All notable changes to IntraMind AI will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-07-02

### Added
- Agent Orchestrator with tool-calling support via Spring AI
- RAG Service with PGVector for semantic document search
- Multi-format document ingestion (PDF, Word, Markdown, HTML, TXT)
- Knowledge graph auto-construction from ingested documents
- Meeting minutes AI summarization with speaker diarization
- Docker Compose one-click deployment (PostgreSQL + Redis + MinIO)
- RESTful API with Spring Boot 3.x
- Ollama local model integration (default: qwen2.5:7b)
- Role-based access control (Enterprise edition)
- Audit logging with citation enforcement (Enterprise edition)
- Open-core licensing model (Community: Apache-2.0)

### Documentation
- Architecture design document with data flow diagrams
- API reference documentation
- Deployment guide (Docker + bare-metal)
- Security best practices guide
- Demo script for quick evaluation
- Open-core feature comparison table
